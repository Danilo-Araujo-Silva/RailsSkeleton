# Rails Skeleton

A Ruby on Rails skeleton application proposal.

## Installation

- gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3
- \curl -sSL https://get.rvm.io | bash
- rvm install ruby --latest
- gem install rails
- gem update
